CSS Position 属性演示

1. position: static（默认）

static

静态定位是每个元素的默认值，元素出现在正常的文档流中

2. position: relative

relative

相对定位，相对于元素的正常位置进行定位

3. position: absolute

absolute

绝对定位，相对于最近的已定位父元素进行定位

4. position: fixed

固定定位，相对于浏览器窗口进行定位，即使页面滚动也不会移动

fixed

5. position: sticky

滚动一下看看效果：

sticky

粘性定位，基于用户的滚动位置进行定位
